PHP Micrososft Translator Library
=================================

Provides two relatively small library files to allow you to access and query the Microsoft Translator API.

Software requirements
---------------------

* PHP 5.x
* PHP-SOAP (any)
* PHP-CURL (any)

History
-------

Written for personal use, and released on GitHub to provide others with a PHP solution.

Notes
-----

The AzureMarketplaceAuthenticator class can be used to access any of the Azure Marketplace offerings that require basic OAuth authentication.
